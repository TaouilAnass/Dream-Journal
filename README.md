Dream Journal – AI-Powered Dream Visualization

Dream Journal is a web application that allows users to capture, visualize, and organize their dreams with the help of AI. Users can describe their dreams, assign moods, and see them transformed into unique images generated by an AI API.

Built with React on the frontend and Node.js/Express on the backend, this app offers a smooth, interactive, and visually appealing dream journaling experience.

Features

AI Dream Visualization: Transform textual dream descriptions into stunning images.

Mood Tagging: Tag dreams with moods like Peaceful, Scary, Happy, Sad, or Adventurous.

Dream Gallery: View your past dreams with images, moods, and timestamps.

Download & Delete: Easily download dream images or remove entries.

Persistent Storage: Dreams and theme preferences are saved in localStorage.

Dark & Light Mode: Toggle between dark and light themes for a personalized experience.

Installation

Clone the repository

git clone https://github.com/your-username/dream-journal.git
cd dream-journal


Install dependencies

# Frontend
cd client
npm install

# Backend
cd ../server
npm install


Run the app

# Start backend server
npm run start

# Start frontend (in another terminal)
cd ../client
npm start


Open http://localhost:3000
 in your browser.

API Configuration

Ensure your Node/Express backend is running on port 5000.

The frontend expects a POST endpoint at /api/generate that accepts:

{
  "prompt": "Your dream description here"
}


The API should return a JSON object:

{
  "image": "url-to-generated-image"
}

Usage

Enter your dream description in the textarea.

Optionally select a mood.

Click Generate Dream Image.

View generated images in your Dream Gallery.

Download or delete any dream as needed.

Toggle between dark and light mode using the theme button.
